# 👩‍💼 HR Data Analysis & Visualization – Excel Dashboard  

## 📌 Project Overview  
This project presents an interactive **HR Analytics Dashboard** built using **Microsoft Excel** to analyze employee data and generate meaningful workforce insights.  

The dashboard focuses on understanding **employee demographics, attrition, salary distribution, performance, and recruitment trends**, helping organizations make better HR decisions.

## 🎯 Objectives  

- Analyze overall **employee distribution and workforce size**  
- Understand **attrition trends across departments**  
- Evaluate **salary and experience patterns**  
- Study **gender and marital status distribution**  
- Identify **recruitment sources effectiveness**  
- Analyze **employee satisfaction and absences**
